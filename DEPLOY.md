# Развёртывание для агента-оценщика

## Вариант 0 — online-демо (без установки)

Развёрнутое приложение доступно по адресу:
**https://oncology-assistant.streamlit.app**

Можно проверить функциональность сразу, без локальной установки. Запросы обрабатываются настоящим экземпляром Yandex AI Studio и реальным PubMed E-utilities.

Если требуется локальный запуск или проверка воспроизводимости — следуйте инструкциям ниже.

---

Все шаги воспроизводимы без ручных действий, кроме создания файла `.env` с API-ключом.

## Предусловия

- Docker Engine 24+ с Docker Compose v2
- Действующий API-ключ от Yandex AI Studio
- Folder ID каталога Yandex Cloud

## Шаг 1 — настройка переменных окружения

Создайте файл `.env` в корне репозитория:

    YANDEX_API_KEY=<ваш_api_ключ>
    YANDEX_FOLDER_ID=<ваш_folder_id>

Альтернативно, скопируйте из шаблона:

    cp .env.example .env
    # отредактируйте .env, подставив свои значения

API-ключ создаётся в Yandex Cloud Console:
https://console.yandex.cloud → AI Studio → Создать API-ключ

## Шаг 2 — сборка контейнера

    docker compose build

Первая сборка занимает ~30-60 секунд (зависимости: openai, requests, python-dotenv, streamlit).

## Шаг 3 — запуск веб-приложения

    docker compose up

Веб-интерфейс будет доступен на:
http://localhost:8501

Чтобы остановить — Ctrl+C в терминале.

## Шаг 4 — запуск в фоне (для длительных демонстраций)

    docker compose up -d

Просмотр логов:

    docker compose logs -f

Остановка:

    docker compose down

## Альтернативный запуск без Docker

Требуется Python 3.11:

    python3.11 -m venv venv
    source venv/bin/activate
    pip install -r requirements.txt
    streamlit run src/web/app.py

## Проверка работоспособности

После запуска:

1. Открыть http://localhost:8501
2. Ввести тестовый вопрос: *Какова эффективность osimertinib при NSCLC с мутацией EGFR T790M после прогрессии на гефитинибе?*
3. Нажать "Запустить анализ"
4. Ожидать 1-2 минуты — должны появиться разделы: PICO, Scoping Review, Найденные источники, Возможность метаанализа, Финальный отчёт

## Критерии «всё работает»

- [ ] `docker compose build` завершается без ошибок (~3-4 минуты)
- [ ] `docker compose up` запускает Streamlit-сервер на порту 8501
- [ ] Веб-интерфейс открывается в браузере без ошибок
- [ ] Тестовый запрос проходит весь пайплайн (6 шагов: Verifier → Researcher → Scoping → Critic → Meta-Checker → Synthesizer) и завершается формированием отчёта
- [ ] В отчёте присутствуют реальные PMID из PubMed
- [ ] Раздел "Возможность метаанализа" корректно классифицирует возможность

## Возможные проблемы

**Ошибка `YANDEX_API_KEY не найден в окружении`**
→ Убедитесь что файл `.env` создан в корне репозитория и содержит обе переменные.

**Ошибка 503 от Yandex AI Studio**
→ Временная перегрузка сервиса. Подождать 30-60 секунд и повторить.

**`docker compose: command not found`**
→ Установлена устаревшая версия Docker Compose. Используйте `docker-compose` (через дефис) или обновите Docker Desktop.

**Долгая первая сборка**
→ Зависимостей мало (~100MB). Если сборка идёт долго — проверьте интернет.

## Остановка и очистка

    docker compose down --rmi local -v

Полная очистка с удалением образов и томов.
