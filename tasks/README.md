# Задания хакатона (справочник)

Папка `tasks/` — **копии всех заданий** с **открытыми критериями оценки** для справки команды и агента-оценщика.

Закрытые критерии (детальные шкалы баллов) командам **не выдаются** — они используются жюри и ИИ-судьёй.

| Файл | Трек |
|------|------|
| [task-01.md](task-01.md) | Виртуальная гистотопограмма |
| [task-02.md](task-02.md) | Прогноз заболеваемости и выживаемости |
| [task-03.md](task-03.md) | AI-ассистент для научных исследований |

Выбранное задание указывается в [README.md](../README.md) и [docs/overview.md](../docs/overview.md).
